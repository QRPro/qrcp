Quick Resto - Content Provider Mapper
========

Content Provider mapper for Android by Quick Resto, Inc.

## Download

Gradle:
```groovy
repositories {
    mavenCentral()
    maven { url "https://jitpack.io" }
}

compile 'com.github.skornei:qrcp:1.0.0+'
```
